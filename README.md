# Running
1. Install rustc
2. run `cargo run` (optionally add `--release` flag)

# Keybinds
| Key | Function |
| --- | -------- |
| Esc | quit     |
| h   | move 1 unit left |
| j   | move 1 unit down |
| k   | move 1 unit up |
| l   | move 1 unit right |
| Shift-h | move 10 units left |
| Shift-j | move 10 units down |
| Shift-k | move 10 units up |
| Shift-l | move 10 units right |
| Space | pause/unpause |
| x | set/unset field |
